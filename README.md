# Obstacle_Avoiding_Robot
Created a obstacle avoiding robot using Arduino for university lab project.

Component used in this project are:
	- 1 Arduino R3
	- 1 L298N H-Bridge dual motor driver
 	- 2 DC motor
        - 1 servo motor
        - 1 Sonar Sensor (HC-SR04)
        - 1 Robot Chassis
        - 4 Wheels
